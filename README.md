# 限制
#### 类型
* Text
* Number
* Currency
* Picklist
* Date 
* DateTime
* Phone
* ...  
#### 当前支持的选项
* API
* 标签
* 类型
* 长度（格式如：文本255 数字/货币:16,2）
* 选项列表值(格式如:A;B;C)
* 公式（填写公式内容）
